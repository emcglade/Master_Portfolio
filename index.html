ETL Process for Atlas - Global Monitoring Tool

An ETL process to collect switch logs from Options global switch infrastructure, tranform the logs ingesting them via their event time, ensuring every log had an event time.
I make sure each log is collected and covereted into a UTC time format.Why was this important for this project?
The syslog boxes where the data is collected from are all in different time zones in the cluster - putting them into a universal time format will keep time standardised.
Once I have the transformed data using Python Pandas, I use the MongoDB python module to send the logs to individual collections, ready to be visualised and displayed on Options Website.
